# Advanced BMI Calculator with History & Trend Analysis

This is a desktop BMI (Body Mass Index) calculator built with Python and Tkinter. It provides a user-friendly interface to not only calculate BMI but also to track and visualize your progress over time.



## ✨ Features

- **Standard BMI Calculation**: Calculates BMI from user input for weight (kg) and height (m).
- **Instant Feedback**: Provides immediate BMI classification (e.g., Underweight, Normal weight, Overweight) with color-coded results.
- **Persistent History**: Automatically saves every calculation to a local SQLite database.
- **Full History Log**: Displays a complete, scrollable log of all past entries.
- **Personal Trend Analysis**: Generates a visual graph that plots a user's BMI changes over time.

## 🛠️ Setup and Installation

**1. Install the required libraries:**
Open your terminal and run this command:
```bash
pip install -r requirements.txt
```

**🚀 How to Run**

After installation, run the main application file from your terminal:
```bash
python bmi_calculator.py

```
